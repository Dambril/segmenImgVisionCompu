# segmenImgVisionCompu
Este proyecto realiza la segmentación de una imagen utilizando técnicas de procesamiento digital como:
* Corrección gamma
* Suavizado gaussiano
* Umbralización
* Operaciones morfológicas
* Detección de contornos

Requisitos
* Python 3
* OpenCV
* Numpy
* Matplotlib

## Resultados
El programa muestra:
* Imagen original
* Resultados del preprocesamiento
* Resultados de los métodos de segmentación
* Aplicación de operaciones morfológicas
* Máscara generada
* Imagen segmentada
